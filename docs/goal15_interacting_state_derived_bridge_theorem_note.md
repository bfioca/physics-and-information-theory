# Goal 15: Interacting State-Derived Bridge Theorem

Goal 15 upgrades the finite bridge benchmarks from product encoded-mouth
resources to a non-product interacting stabilizer resource. The model dresses
the right encoded mouths by a logical `CZ` interaction graph. This makes the
named bridge components mutually correlated while keeping exact stabilizer and
channel certificates available.

## Theorem Package

The result has two parts.

**Positive theorem.** A logical-`CZ`-dressed encoded bridge is a non-product
finite stabilizer state whose observer algebra and mouth map are recovered from
the same interacting state. Full encoded-block mutual information infers the
mouth map, while low-order physical entropy through the code distance remains
blind. Removing the inferred interaction and routing by the inferred mouth map
gives exact bridge transfer. Wrong-mouth and mouth-blind controls fail.

**No-go theorem.** The same static non-product state does not determine a
screen recovery transition. Opposite north- and south-favored screen dynamics
can share the same static entropy, observer algebra, and bridge-map signature.
The minimal missing ingredient is an explicit screen dynamics/isometry, or an
equivalent rule assigning the bridge state to recovery channels.

## Representative Witness

The default witness uses two left mouths and two right mouths encoded in
independent `[[5,1,3]]` blocks, then applies one logical `CZ` between the right
encoded mouths.

| Diagnostic | Result |
| --- | --- |
| non-product bridge witness | bridge-component MI rises from `0` to `2` |
| inferred interaction graph | `((0,1),)` from inter-bridge MI excess |
| inferred twisted mouth map | `(1,0)` from full-block MI |
| low-order entropy through order 3 | aligned/twisted match |
| first entropy mismatch | order `4` |
| identity activation after interaction removal | `0` qubits for twisted witness |
| state-derived activation | `2` qubits |
| state-derived `T`-dressed activation | `2` qubits |
| mouth-blind scrambling control | `0` qubits |

The dressed observer algebra is generated by the right-block logical Pauli
algebra conjugated by the inferred logical `CZ` graph. For the two-mouth
witness its logical dimension is `4`, symplectic rank is `4`, and center
dimension is `0`.

## Transition Boundary

The theorem deliberately does not pretend that a static state alone supplies a
north/south recovery dynamics. The certificate records two completions with the
same static-state signature:

- a north-favored screen channel;
- a south-favored screen channel.

They have opposite recovery winners. This proves the static-state no-go inside
the declared finite model class. Once a screen isometry is explicitly included,
the Goal 14 channel-derived transition theorem applies: success probabilities
are read from the induced screen channels, and the equal-area entropy analogue
matches the recovery ordering.

## Bounded Atlas

The bounded atlas checks all encoded-mouth pairings for `m <= 3` under the
complete logical-`CZ` right-mouth interaction graph.

- `m=1`: no non-product interaction is possible.
- `m=2`: first non-product entropy-matched transfer split.
- `m=3`: all pairings are recovered from the interacting state; state-derived
  activation has full capacity for every pairing; mouth-blind scrambling has
  zero structured capacity.

## Claim Boundary

This is an exact finite stabilizer/QEC theorem plus dense Choi/Kraus channel
certificate. It is not a continuum ER=EPR theorem, not de Sitter physics, not
dS/CFT, and not a chaotic many-body traversable-wormhole simulation.

The result says something narrower and useful: an interacting finite bridge
state can determine its observer algebra and bridge map operationally, but a
screen recovery transition needs dynamics, not static state data alone.
