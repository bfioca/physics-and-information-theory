# Static Quadrupole Master Source

This note fixes the missing normalization and derives the exact map from a
conserved static `ell=2` stress to the pure-de-Sitter Zerilli-Moncrief source.
It also treats the distribution jets generated by a moving thin membrane.

## Conventions

Use signature `(-+++)`,

```text
N=1-r^2/R^2,
delta(G^mu_nu+Lambda delta^mu_nu)=kappa delta T^mu_nu,
kappa=8 pi G,
```

and Regge-Wheeler gauge

```text
h_tt=N H0 Y,
h_rr=H2 Y/N,
h_AB=r^2 K Y gamma_AB.
```

The fixed `ell=2` Zerilli-Moncrief normalization is

```text
Psi=r/3 [K+N(H2-rK')/2].
```

The source amplitudes use

```text
delta T^t_t=-rho Y,
delta T^r_r=p_r Y,
delta T^r_A=j D_A Y,
delta T^A_B=p_perp Y delta^A_B+pi Y^A_B.
```

## Einstein Elimination

A direct component calculation gives the exact identity

```text
A_2 Psi=(r^2N/6)(delta G^t_t)'
        -r(1+4r^2/R^2)delta G^t_t/6
        -r delta G^r_r/2-delta G^r_A+2r delta G_TF,

A_2=-(N Psi')'+6Psi/r^2.
```

Therefore

```text
F=kappa[-r^2 N rho'/6
        +r(1+4r^2/R^2)rho/6
        -r p_r/2-j+2r pi].
```

This formula is insensitive to the normalization of `Y` provided all stress
amplitudes use the same harmonic. For the repository tensor
`q=Q_ab n_a n_b`,

```text
integral_S2 q^2 dOmega=(8pi/15) Tr(Q^2).
```

For `Q=Omega Omega-delta Omega^2/3`, this is `16pi Omega^4/45`; `q` is not a
unit-normalized spherical harmonic.

## Thin-Shell Jets

Let the canonical stress contain

```text
rho=rho_b Theta(a-r)+rho_0 delta+rho_1 delta',
p_r=p_b Theta+p_0 delta+p_1 delta',
j=j_b Theta+j_0 delta+j_1 delta',
pi=pi_b Theta+pi_0 delta+pi_1 delta'.
```

Define

```text
A=r^2N/6,
B=r(1+4r^2/R^2)/6.
```

Then the literal metric-defined master source is

```text

D_2=-kappa A rho_1,
D_1=kappa[-A rho_0+2A' rho_1+B rho_1
           -a p_1/2-j_1+2a pi_1],
D_0=kappa[A rho_b(a)+A' rho_0-A'' rho_1
           +B rho_0-B' rho_1-a p_0/2+p_1/2
           -j_0+2a pi_0-2pi_1].
```

The `delta''` term is a contact artifact of applying the vacuum-defined master
variable to an infinitesimally thin displaced shell. Write

```text
Psi=Psi_off+c delta,
c=-D_2/N_a.
```

The equivalent contact-free source for off-wall observables is

```text

```

For `r!=a`, its response is

```text

           +D_0^off G_2(r,a)-D_1^off partial_a G_2(r,a).
```

Consequently the existing `L2` and total-variation source bounds do not apply
directly to the ideal membrane. One must use the contact-free transmission
problem or a finite-thickness regulator.
