# Interacting Symmetry-Preserving KMS Edge Model

Status: exact finite boundary interaction plus theorem-backed Type-III/core limit

## Model

The factorized core theorem made the thermal sector a spectator. The next
controlled step couples it to the fuzzy harmonic energy. At cutoff `L`, let the
boundary consist of the harmonic space and the first thermal qubit, with

```text
H_B = a K_L + Delta n_1 + g K_L n_1.
```

For `L>=1` and `beta>0`, the state is the faithful Gibbs state
`rho_B proportional exp(-beta H_B)`.
The remaining infinite qubit tail keeps the alternating gaps `1,sqrt(2)` and
their product Gibbs state at the same inverse temperature. The declared
C*-dynamics is the boundary Heisenberg dynamics generated by `H_B`, tensored
with the onsite tail dynamics; the product state is beta-KMS for this dynamics.

For the declared domain `g>0`, the first-site excitation probability depends on `ell`, so the
boundary state does not factorize between angular energy and the thermal qubit.
Analytically, `I(ell:n_1)` is strictly positive. The implementation evaluates it
with log-sum-exp normalization and records log probabilities so extreme Gibbs
weights cannot be mistaken for a nonfaithful mathematical state. This is an
interacting, though commuting and classically correlated, boundary state rather
than the previous tensor-product spectator construction.

## Symmetry And Algebra Type

The interaction depends only on the Casimir `K_L`. It commutes with `J_z` and
the full `SU(2)` action, so magnetic degeneracy is protected. The time,
time-then-axial, and full-rotation expectations preserve the Gibbs state and
commute with its modular flow.

Only the first thermal site is absorbed into the finite boundary factor. The
tail still contains both gap families infinitely often, so its product-state
factor remains hyperfinite Type-`III_1`. Hence

```text
M_{2(L+1)^2} tensor R_tail
```

is Type-`III_1`. The boundary modular action is inner. Exterior equivalence via
the boundary-density cocycle, relative to the declared tensor split and trace
convention, identifies the core with

```text
M_{2(L+1)^2} tensor C_tail,
```

which is hyperfinite Type-`II_infinity`. The symmetry expectations preserve
matrix trace and the boundary density, hence fix the cocycle and extend to
normal trace-preserving expectations on this core.

## Interacting Core Obstruction

Fix the `ell=L` sector. Since `K_L` is scalar on `V_L`, the conditional thermal
density is identical for every magnetic state. Normal phase/orientation probe
states can therefore be tensored with the same first-site and core density. The
interaction does not change the exact probe conclusions:

```text
decoder error >= 1/2,
time-then-axial relative-entropy loss = log 2,
full-SU(2) relative-entropy loss = log(2L+1).
```

The KMS background itself is invariant under every declared expectation, so its
relative-entropy loss is zero. The nonzero `log 2` and `log(2L+1)` values belong
to normal probes prepared in the KMS representation. This is the first model in
the program with a non-product angular-thermal Gibbs background, a protected
rotation action, a Type-`III_1` limit, a Type-`II_infinity` core, and a probe
edge obstruction in the same construction.

## Universal Rotational-Scalar Bath No-Go

The one-qubit calculation is an example of a bath-independent theorem. Suppose
the bath carries the trivial rotation action. On a fixed irreducible sector
`V_L`, every `SU(2)`-invariant Hamiltonian has the Schur form

```text
H|_{V_L tensor K_L} = I_{V_L} tensor H_{bath,L}.
```

Consequently, the conditional Gibbs background is maximally mixed on `V_L`,
tensored with an arbitrary bath state, and has zero loss under rotation
averaging. Neither interaction strength, bath Hamiltonian, Type-III factor
structure, nor the Type-II clock core changes the probe laws

```text
decoder error >= 1/2,
D_time+axial = log 2,
D_full-SU2 = log(2L+1).
```

A rotationally scalar bath cannot become the missing orientation reference.
The only escape routes are to give the bath nontrivial rotation charge/reference
data, break the symmetry, or change the angular representation content. This is
the sharp discriminator supplied by the interacting surrogate.

## Boundary

The interaction is still engineered and confined to one bath qubit. There is
no local relativistic field, de Sitter static-patch metric, Bunch-Davies state,
stress tensor, horizon area term, or generalized-entropy identity. The result
is a controlled mathematical-physics bridge, not a gravitational breakthrough.

The next gate is to replace the single qubit by a local field or spin net whose
KMS state and rotation action follow from a geometric Hamiltonian. The
