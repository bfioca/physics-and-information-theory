# Finite Type-Certification Prove-or-Kill Goal

Status: completed on 2026-06-17. The binding outcome is
`docs/finite_type_certification_control_result.md`.

Determine whether an observer with bounded probe dimension, channel calls,
energy, duration, localization, and adaptive memory can operationally certify
non-Type-I algebraic structure or algebraic connectivity, or whether every
such protocol is reproducible by an admissible Type-I surrogate. First fix the
protocol class and output metric. Then derive the strongest control already
implied by hyperfiniteness, nuclear completely positive approximation, split
inclusions, and energy-constrained channel approximation. A surrogate must
preserve the declared finite local interface, state, covariance, dynamics,
resource ledger, and refinement maps; reproducing one transcript or one finite
channel is not a theorem.

Treat this as a prove-or-kill sprint. Run exact finite models and
counterexamples before attempting a general proof. Stop if the candidate is a
standard approximation corollary, if the operational distance cannot be
controlled, or if preserving the complete global structure makes the premise
false by definition. Continue toward a paper only if a genuinely new residual
survives: an explicit lower or upper resource-scaling theorem showing which
physical resource must diverge before factor type or connectivity becomes
certifiable. Deliver a primary-source novelty audit, a precise control theorem,
reproducible finite examples, and a candid GO/STOP decision without delaying
Paper A.

## Completion Decision

The generic theorem stopped. Type classification already has operational
entanglement characterizations, while any fixed finite cylinder in the
repository's hyperfinite model has an exact Type-I realization. The retained
research residual is narrower and physical: constrain the admissible local
unitaries in a named Hamiltonian or QFT model and bound their support, energy,
time, complexity, or gravitational cost as the type witness becomes accurate.
