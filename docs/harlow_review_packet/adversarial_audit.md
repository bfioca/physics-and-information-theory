## Corrections Made

1. Replaced “directional-record information” with the narrower
   “operational rotational-asymmetry resource” in the packet, review question,
   email, and diagram.
2. Replaced the implication that a completed-source interval already exists
   with the accurate statement that no rigorous continuum interval for the
   completed source presently excludes zero.
3. Narrowed the Skyrmion claim from making the realization problem nonempty to
   making only the supported-profile subproblem nonempty.
4. Made the logical status of `B_W` explicit: only a nonzero value would witness
   non-invisibility, and zero exclusion remains open.

  non-invisibility only. It would not close the common-action channel,
