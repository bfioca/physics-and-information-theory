# Phase 1 Physical Model Specification

Status: finite geometric regulator and declared-atlas observer experiment
implemented; general inverse theorem and gravitational lift remain open

## Purpose

This document freezes the first replacement model for the research program. It
uses the standard fuzzy two-sphere as a finite noncommutative geometric
regulator. The construction is established mathematics. The research question
is what happens when a physical observer experiment, noisy algebra inference,
and canonical refinement are built on top of it.

## Kinematics

At integer level `L >= 0`, set

```text
j = L/2,
H_L = C^{L+1},
A_L = B(H_L) = M_{L+1}.
```

`H_L` is the spin-`j` representation of `SU(2)`. The fuzzy function algebra is
`A_L`. Its vector-space dimension is `(L+1)^2`.

This replaces the earlier ambiguous construction in which `(L+1)^2`
spherical harmonics were treated as Hilbert-space states and then quantized
again into `M_{(L+1)^2}`.

## Symmetry And Coordinates

Hermitian generators satisfy

```text
[J_a,J_b] = i epsilon_abc J_c,
sum_a J_a^2 = j(j+1) I.
```

For `L>0`, normalized coordinates are

```text
X_a = J_a / sqrt(j(j+1)),
sum_a X_a^2 = I.
```

`SU(2)` acts on observables by conjugation. Infinitesimally,

```text
mathcal J_a(A) = [J_a,A].
```

No axis-dependent `m` splitting is used in the primary regulator.

## Fuzzy Harmonics

The adjoint representation decomposes as

```text
A_L = direct_sum_{ell=0}^L V_ell.
```

Each spin `ell` occurs once, so

```text
sum_{ell=0}^L (2 ell + 1) = (L+1)^2.
```

The implementation constructs normalized tensor harmonics `T_{ell,m}` from
the highest-weight operator `J_+^ell` and adjoint lowering by `J_-`. The frozen
conventions are

```text
tau_L(A) = Tr(A)/(L+1),
tau_L(T_{ell,m}^* T_{ell',m'}) = delta_{ell,ell'} delta_{m,m'},
T_{ell,m}^* = (-1)^m T_{ell,-m},
T_{0,0} = I.
```

These harmonics are operator multipoles. They are not states in `H_L`.

## State And Dynamics

The reference state is the normalized trace `tau_L`. This is the unique
`SU(2)`-invariant tracial state on `A_L`; it is a regulator reference state, not
yet a gravitational static-patch state.

The positive fuzzy Laplacian is

```text
Delta_L(A) = sum_a [J_a,[J_a,A]],
Delta_L(T_{ell,m}) = ell(ell+1) T_{ell,m}.
```

The heat channel is

```text
P_t^{(L)} = exp(-t Delta_L).
```

It is computed in the complete harmonic basis. The implementation directly
checks unitality, trace and adjoint preservation, the semigroup law, covariance,
and Choi positivity at finite levels.

## Low-Mode Observable System

For `K <= L`, define

```text
V_K^{(L)} = span{T_{ell,m}: 0 <= ell <= K}.
```

This is an `SU(2)`-invariant unital self-adjoint operator system. It is generally
not an algebra because products of modes through `K` can contain angular
momenta through `2K`.

The implementation supplies the Hilbert-Schmidt projection onto
`V_K^{(L)}`. That projection is a kinematic comparison tool. It is not claimed
to be positive or to be the physical screen channel.

## Observer Experiment Boundary

The fuzzy sphere by itself does not define an observer or a horizon
measurement restriction. Phase 1 still requires a physical quantum statistical
experiment. Candidate constructions include:

- a coherent-state POVM/Berezin symbol experiment;
- a finite angular-resolution covariant POVM;
- an operator system generated by measured multipoles and response probes;
- or a conditional expectation derived from a later constraint/observer model.

The selected experiment must be completely positive at the appropriate level,
must declare its state family and decision tasks, and must not be defined merely
to erase the response that will later be rediscovered.

### Implemented coherent-state experiment

The first screen experiment is the canonical spin-coherent POVM

```text
dM_L(n) = (L+1) |n><n| dOmega/(4 pi).
```

`qgtoy.fuzzy_screen` realizes it with Gauss-Legendre and uniform-azimuth
quadrature. The finite effects sum to the identity to numerical precision and
span `M_{L+1}` linearly. Thus the measurement is tomographically complete as a
many-copy statistical model; it is not a diagonal erasure chosen to force a
collision.

For a single copy, however, the fixed classical measurement is less informative
than the full quantum experiment for some decision tasks. The implemented pole-
state test compares full Helstrom discrimination with total-variation distance
of the coherent-screen distributions. If one classical-to-quantum recovery map
reconstructs both states within trace distance `epsilon`, contractivity gives

```text
epsilon >= (D_quantum - TV_screen)/2.
```

This is the first operational screen-deficiency witness. It is a finite binary
decision bound, not yet the unknown-maximal-algebra theorem.

### Implemented finite inverse benchmark

`qgtoy.fuzzy_algebra_inference` places one of three declared channels before
the coherent screen: the identity, complete `J_z` dephasing, or complete
depolarization. Mixed `ell=1` direction probes `(I+/-J_z/j)/(L+1)` and
`(I+/-J_x/j)/(L+1)` give pairwise separated two-response signatures. If each response is known within
`epsilon < Delta_L/2`, where `Delta_L` is the minimum signature separation,
the candidate is uniquely identified. A Hoeffding bound supplies an explicit
sample count.

For each candidate, the code separately verifies identity recovery on the
standard exact OA-QEC algebra `M_{L+1}`, diagonal `C^{L+1}`, or scalar `C I`.
This is a finite model-selection theorem inside a fixed atlas. It is not yet an
estimator for an unknown arbitrary Wedderburn algebra, and the coherent screen
is not claimed to recover a noncommutative state from one copy.

The direction-witness response is exactly `1/3` at every cutoff. Both probe and
witness transport under the Berezin maps with explicit inverse-cutoff error.
This removes the previous
dependence on high-mode rank-one probes from the continuum research path.

## Canonical Cutoff Refinement

For `L <= M`, the selected refinement is the coherent-state Berezin map

```text
J_{L->M}=Q_M sigma_L.
```

It is UCP, normalized-trace preserving, `SU(2)` equivariant, and canonical for
the frozen coherent-state regulator. On normalized harmonics it has the exact
multiplier `sqrt(b_{L,ell}b_{M,ell})`, which tends to one at fixed `ell` when
both cutoffs grow, with an explicit inverse-cutoff bound. It exactly
intertwines the fuzzy Laplacian and heat flow. Its cutoff-composition defect
also vanishes at fixed `ell`.

The map is entanglement breaking. No global full-matrix multiplicativity claim
is made. On the complete coefficient-atomic unit ball of
`span{I,X_x,X_y,X_z}`, the adjacent-cutoff product defect is at most
`8/[3(L+2)]`. Conversely, a highest-mode partial isometry gives a global
unit-ball defect tending to `1/2`, proving that the low-mode restriction is
necessary.

## Directly Verified In Code

`qgtoy.fuzzy_sphere` computes:

- the spin generators, Casimir, and sphere relation;
- the complete harmonic basis and full Gram matrix;
- weight, raising, lowering, conjugation, and trace-sector identities;
- the full adjoint-Laplacian spectrum;
- Dirichlet positivity on every matrix unit;

## Explicit Nonclaims

This finite model does not yet provide:

- a gravitational constraint or observer clock;
- the CLPW de Sitter observer algebra;
- a physical screen channel;
- a product theorem on a growing harmonic domain beyond the proved coordinate
  operator system;
- arbitrary-channel noisy maximal-algebra identification;
- a Type-II limit;
- generalized entropy;
- or a de Sitter prediction.

Those are subsequent gates, not properties inferred from the fuzzy-sphere
kinematics.
