# Charged Invariant-Subsystem Encoding

Status: exact finite singlet-code baseline with explicit noncovariance boundary

## Two Different Objects

Two useful constructions can be placed beside the same spin-`L` sector and a
common spectator KMS/core factor:

```text
fixed-algebra test: B(V_L)^{SU(2)},
singlet code:       V_L tensor V_L^* tensor C^{2L+1}.
```

The second uses the spin-`L` block structure of the Peter-Weyl
representation `L^2(SU(2))`: the conjugate irrep carries rotation charge and
the multiplicity register carries the logical information.

These are not yet a matched operational control and treatment. The first uses
the fixed operator algebra. The second compresses to invariant vectors in the
total-spin-zero sector. Keeping that distinction explicit prevents a singlet
code from being mistaken for a prepared quantum reference frame.

## Scalar Fixed-Algebra Test

A scalar reference does not change the full-rotation fixed algebra on `V_L`:

```text
B(V_L)^{SU(2)} = C I_{V_L}.
```

Orthogonal phase probes collide. Every decoder has worst-case trace-distance
error at least `1/2`, and a pure probe has orientation deficit `log(2L+1)`.

## Charged Invariant Code

Let `d=2L+1`, `R_charge=V_L^*`, and let the reference multiplicity register
`K` have dimension `d`. Define

```text
|Omega_L> = sum_i |i>_S |i*>_R / sqrt(d),
W|k> = |Omega_L> tensor |k>_K.
```

`W` is an exact isometry into the invariant subspace of
`V_L tensor V_L^* tensor K`. The invariant subspace is

```text
span{|Omega_L>} tensor K,
```

so its invariant logical algebra is `B(K)=M_d`. The decoder discards the fixed
singlet factors and reads `K`. It recovers every logical matrix unit, hence the
pre-encoded logical channel, with diamond error zero.

Here `L` is restricted by the implementation to positive integer spin. The
representation-theoretic discussion extends to the other `SU(2)` irreps after
reparameterizing the API by twice-spin.

This decoder is abstractly exact but not a covariant decoder back to a charged
spin-`L` output. Every codeword is invariant, so `U_out(g)W=W`. Covariance from
the charged input would require `U_out(g)W=WU_L(g)`, which is impossible for an
injective `W` when `L>0`. A `z` rotation by `pi/L` gives operator-norm
intertwiner defect `2` on a highest-weight input.

The full fixed operator algebra on the charged auxiliary system is

```text
B(V_L tensor V_L^* tensor K)^{SU(2)}
  = direct_sum_{J=0}^{2L} B(K).
```

The displayed `B(K)` is only its singlet-sector corner, not the full fixed
operator algebra.

## Scoped Dimension Bound

Write a general finite reference representation as

```text
R = direct_sum_j V_j tensor C^{m_j}.
```

The invariant subspace of `V_L tensor R` has dimension `m_L`. A pure isometry
of all `d` logical states into total spin zero requires `m_L>=d`. The conjugate
spin-`L` sector therefore contributes at least

```text
dim(V_L^*) m_L >= d^2 = (2L+1)^2
```

auxiliary dimensions. The truncated Peter-Weyl block used above saturates this
bound within the singlet-isometry class.

This is not an unrestricted minimum for exact channels. A trivial
`d`-dimensional register permits the noncovariant channel
`rho -> I_{V_L}/d tensor rho_K`, followed by partial trace. Charge becomes
necessary only after the common physical task and covariance restrictions are
specified.

## Physics Interpretation

This is a representation-theoretic positive baseline for the scalar-bath
no-go. A Type-II clock does not create an invariant logical subsystem by
itself, while an auxiliary system carrying the correct charge and multiplicity
does. Representation content, not merely clock entanglement, determines which
invariant subsystems are available.

The result is not an operational quantum-reference-frame recovery theorem. The
encoding `W` is a global noncovariant pre-encoding of the logical state; it is
not the channel obtained by appending a fixed prepared reference state `eta_R`
to an unknown system state `rho_S` and then applying the `SU(2)` twirl. A
physical reference theorem must specify one common append-and-twirl task, the
allowed covariant encoder/decoder, and the optimal error as a function of
reference charge, dimension, and energy.

Tensoring either finite calculation with the same normalized KMS/core density
preserves its distances and abstract recovery identities. That is spectator
tensor stability only; the charged code has not been generated by the
interacting KMS dynamics or its modular core.

The result also remains a selected compact-rotation sector. It does not include
de Sitter boosts, a local field Hamiltonian, the energetic cost of the
reference, or a Bunch-Davies/static-patch derivation.
